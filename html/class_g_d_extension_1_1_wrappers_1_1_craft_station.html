◆ Bind()

new CraftStation GDExtension.Wrappers.CraftStation.Bind ( GodotObject godotObject)
inlinestatic

Try to cast the script on the supplied godotObject to the CraftStation wrapper type, if no script has attached to the type, or the script attached to the type does not inherit the CraftStation wrapper type, a new instance of the CraftStation wrapper script will get attaches to the godotObject .

The developer should only supply the godotObject that represents the correct underlying GDExtension type.

Parameters
godotObjectThe godotObject that represents the correct underlying GDExtension type.
Returns
The existing or a new instance of the CraftStation wrapper script attached to the supplied godotObject .
