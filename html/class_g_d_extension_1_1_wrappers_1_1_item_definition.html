ItemDefinition GDExtension.Wrappers.ItemDefinition.Bind ( GodotObject godotObject)
inlinestatic

Try to cast the script on the supplied godotObject to the ItemDefinition wrapper type, if no script has attached to the type, or the script attached to the type does not inherit the ItemDefinition wrapper type, a new instance of the ItemDefinition wrapper script will get attaches to the godotObject .

The developer should only supply the godotObject that represents the correct underlying GDExtension type.

Parameters
godotObjectThe godotObject that represents the correct underlying GDExtension type.
Returns
The existing or a new instance of the ItemDefinition wrapper script attached to the supplied godotObject .
