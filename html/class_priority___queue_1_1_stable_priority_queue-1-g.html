Detailed Description

A copy of FastPriorityQueue which is also stable - that is, when two nodes are enqueued with the same priority, they are always dequeued in the same order. See https://github.com/BlueRaja/High-Speed-Priority-Queue-for-C-Sharp/wiki/Getting-Started for more information.
