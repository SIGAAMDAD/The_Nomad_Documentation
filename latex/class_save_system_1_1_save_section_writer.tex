\doxysubsection{Detailed Description}
Handles writing of savefile "{}sections"{} into a binary format. 

Not thread-\/safe. Should ONLY be used in a dedicated Save() function. Exceptions are caught in \doxylink{class_archive_system_a9962d0150f259a457de4fa070e988e68}{Archive\+System.\+Save\+Game}
