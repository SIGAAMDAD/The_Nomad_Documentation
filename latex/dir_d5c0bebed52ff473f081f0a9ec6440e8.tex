Adds new Range control Spinner, an all-\/purpose process indicator. Use to provide feedback on ongoing processes -\/ threads, network requests, timers, etc.\hypertarget{README.md_autotoc_md466}{}\doxysubsubsection{\texorpdfstring{Features}{Features}}\label{README.md_autotoc_md466}

\begin{DoxyItemize}
\item 5 statuses
\begin{DoxyItemize}
\item {\ttfamily EMPTY} for a process that hasn\textquotesingle{}t started
\item {\ttfamily SPINNING} for indeterminate progress, such as a network request
\item {\ttfamily PROGRESSING} for determinate progress, such as a timer or download with known message body size
\item {\ttfamily SUCCESS} for a process that has completed successfully
\item {\ttfamily WARNING} for a process that has completed successfully with warnings
\item {\ttfamily ERROR} for a process that has errored out
\end{DoxyItemize}
\item Can use custom icons, with defaults taken from Godot\textquotesingle{}s own icons
\item Option to use no icon and fills with status color instead
\item Configurable spinner/border width and speed
\item Configurable colors and option to use editor theme
\end{DoxyItemize}\hypertarget{README.md_autotoc_md467}{}\doxysubsubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md467}

Add the Spinner node to your scene. All options update a live preview in the editor. All options are documented in the Inspector panel. If anything isn\textquotesingle{}t clear enough there, open an issue.\hypertarget{README.md_autotoc_md469}{}\doxysubsubsubsection{\texorpdfstring{Basic Usage}{Basic Usage}}\label{README.md_autotoc_md469}
Set {\ttfamily status} to the desired status indication setting. Spinner is a range node, and can have its {\ttfamily min\+\_\+value}, {\ttfamily max\+\_\+value} and {\ttfamily value} updated accordingly, but those values only affect its border fill when {\ttfamily status} is {\ttfamily Status.\+Progressing}.

To set {\ttfamily value} to a new value and {\ttfamily status} to {\ttfamily Status.\+Progressing} at the same time, use {\ttfamily set\+\_\+progressing(value)}. This function can be connected to a signal for easy updates, but be aware that when {\ttfamily value == max\+\_\+value} Spinner will not automatically update to {\ttfamily Status.\+SUCCESS}, that has to be set separately.\hypertarget{README.md_autotoc_md470}{}\doxysubsubsubsection{\texorpdfstring{Borderless Icons}{Borderless Icons}}\label{README.md_autotoc_md470}
If you set {\ttfamily icon\+\_\+borderless}, you probably also want to set {\ttfamily icon\+\_\+scale} to {\ttfamily 1}.\hypertarget{README.md_autotoc_md471}{}\doxysubsubsubsection{\texorpdfstring{Example\+: Watching an HTTPRequest}{Example\+: Watching an HTTPRequest}}\label{README.md_autotoc_md471}
